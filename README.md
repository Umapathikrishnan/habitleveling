

## Local Setup
Install Node.js LTS version [download](https://nodejs.org/en/download/)
1. Create .env.local file in the root directory with below variables

   ```bash
   EXPO_PUBLIC_SUPABASE_URL=your_supabase_url
   EXPO_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```
2. Install dependencies

   ```bash
   npm install
   ```

3. Start the app

   ```bash
   npm start
   ```

Backend is made with supabase.
